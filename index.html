Curriculum

研修カリキュラム

全8回（各2時間・約2ヶ月）の研修で、YouTubeの企画力を内製化し、売上に直結する運用体制を構築します。

W1

立ち上げ

講義オリエンテーション / YouTubeマーケティングの全体像 / 運用者の実務全体像 / チャンネル開設の実践 / 売上（採用）目標のPL設計

W2

チャンネル設計

なぜチャンネル設計が大事なのか？ / チャンネルコンセプト設計 / キャラ設計 / チャンネル設計の実践

W3

企画①

売上（採用）に繋がる企画とは？ / YouTubeリサーチ / 企画・構成・タイトルの考え方 / 台本の書き方 / 1本目の企画完成

W4

企画②

前回の企画・構成・台本のフィードバック / サムネイルの考え方 / 2本目の企画完成

W5

撮影

前回の企画・構成・台本のフィードバック / 撮影環境（場所・機材）/ 撮影ディレクション / 撮影方法と実践

W6

分析

なぜ動画を分析すべきなのか？ / 動画投稿の実践 / YouTube Studioの使い方

W7

LINE構築

リストマーケティングの基本 / 公式LINEの設定方法 / 公式LINE構築の実践

W8

マネジメント

制作スケジュールのマネジメント / 数値管理のマネジメント / マネジメントの実践

研修を終えると、こうなります

企画・チャンネル設計・分析は内製化し、制作はAI・外注へ。
